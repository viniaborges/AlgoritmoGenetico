Algoritmo desenvolvido em 2015

O Problema do Caixeiro Viajante (PCV) é um problema de otimização e decisão, cujo objetivo é encontrar a menor rota para se percorrer uma série de cidades, passando apenas uma vez por cada uma delas e retornando à cidade de origem. Dado um conjunto de n cidades e uma matriz (quadrada de tamanho n) de distâncias, a solução consiste em encontrar a permutação que minimize a rota percorrida. O número de permutações é dado por (n-1)!/2. Então, para um número pequeno de cidades, o problema pode ser rapidamente ser resolvido testando-se todas as permutações, método conhecido como força bruta. Porém, conforme o número de cidades aumenta, o número de permutações aumenta exponencialmente, afetando, como consequência, o tempo de processamento. Com a finalidade de otimizar o tempo de processamento para o problema do PCV com muitas cidades, métodos heurísticos são utilizados para construção e minimização de rotas. Estes métodos podem não fornecer a menor rota, mas se aproximam muito do resultado, quanto melhor for o algoritmo utilizado. O objetivo deste trabalho é estudar o PCV do tipo simétrico e implementá-lo com técnicas de inteligência artificial utilizando algoritmos genéticos (GA). Os resultados deverão ser validados por comparação com benchmarks encontrados na literatura. Este problema (do PCV) foi escolhido estrategicamente por ser de fácil compreensão, adaptável a programação de processos industriais, disponibilidade de muitos dados em literatura, facilitando a validação do algoritmo, além de representar uma ampla classe de problemas para os quais não existem algoritmos polinomiais em séries temporais determinísticos. A representatividade do problema promove flexibilidade e possibilidade de prosseguimento do estudo, aplicando as técnicas estudadas a processos industriais de produção.
